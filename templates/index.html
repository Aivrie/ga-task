Task Introduction

Given the attached dataset, the task is to create one or more data visualization(s) on a web page. The attached dataset was mined from The NASN 2021 Grand Challenge which asks researchers to explore a dataset and share their insights

This task was created by the team at the Collaboratorium for Social Media and Online Behavioral Studies. It was prepared as a small technical evaluation for Okeke Obianuju Ivory. This project is representative of a typical task that may be perform at COSMOS and it ensures the lab meets the student's expectation.

Findings from Analysis of the Dataset

On careful consideration of the dataset, the following analysis below were conducted;

Average Follower Count via Twitter Source

For this analysis, the average follower count of users from each Twitter source was determined.
Selection of relevant Twitter sources were based on the number of followers from each Twitter source.

Correlation between follower count and anger

For this analysis, a scatter plot was used to show the correlation between follower count and the anger emotion.
From the dataset, the anger emotions was described using a range from 1 to 4. The follower count was plotted across the range of anger levels.
